# Geteilte Notizen

Dieser Ordner enthält Notizen, die von allen Benutzern geteilt werden können.

## Verwendung

- In der Anwendung werden hier erstellte Dateien unter "Geteilte Notizen" angezeigt
- Alle authentifizierten Benutzer haben Zugriff (je nach Konfiguration)
- Unterstützte Formate: `.md` (Markdown)

## Produktion

In der Produktionsumgebung wird dieser Ordner typischerweise auf einen NAS-Ordner gemountet:
- Synology: `/volume1/shared` → `/data/shared`
- QNAP: `/share/Public` → `/data/shared`
